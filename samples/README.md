# samples

各スキルを実際に実行して生成したサンプル成果物を収録します。

## ディレクトリ構成

サンプルは **スキル毎のフォルダ** に分けて配置します。今後スキルが増えても整理できるよう、
新しいサンプルは必ず対応するスキル名のフォルダ配下に追加してください。

```
samples/
└── <スキル名>/              例: line-stamp-generator
    └── <サンプル名>/        例: ai-occupied-humans
        ├── README.md        サンプルの説明・再現手順
        └── ...              成果物（画像・metadata など）
```
